Workshops

Workshop Overview


This workshop will be run by Future Friends co-creator Remy Sharp and will look at Modern Universal React dev with Next.js

You will be expected to be comfortable with JavaScript, and be familiar with, or at least comfortable with the JSX concepts (putting XML inside your JavaScript). We'll also be using Node 8 as we'll make use of async/await.

What will be covered


Subject Areas

  • Up and running: application architecture
  • Extending: custom routing, parameters
  • Connecting: Database backed pages
  • Authentication: strategies & techniques
  • Testing, building & deploying

What you will learn


Learning Outcomes

  • How to create & configure Next-based projects
  • Experience with latest JS features, including async/await
  • Exposure to some gotchas
  • Handling special cases around server/client side only & using environment variables
